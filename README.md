Dcat Page
======

`Dcat Page`是一个静态站点生成工具，可以生成个人静态站点和漂亮的Markdown在线文档，采用[laraval.com](https://github.com/laravel/laravel.com)作为基础模板。


## 特性

+ 支持开发环境实时预览
+ 支持文档内容实时搜索
+ 支持二次开发，自定义样式


## 环境
 - PHP >= 7.1.0
 - Laravel >= 5.5.0

## 安装

请先确保安装好`Laravel`框架，然后运行

```php
composer require dcat/page
```
